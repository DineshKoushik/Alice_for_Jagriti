Profit and Loss (लाभ और हानि)

Chapter 3

सबसे पहले हम लाभ एवं हानि ( Profit & Loss ) के बेसिक शब्दावलियों [Basic Terminologies Related to Profit and Loss Maths Chapter)

1. क्रय मूल्य ( Purchasing Price ) : यह वह मूल्य होता है, जिस पर कोई वस्तु खरीदी जाती है.

2. विक्रय मूल्य ( Selling Price ) : विक्रय मूल्य वह मूल्य होता है, जिस पर कोई वस्तु बेची जाती है.

3. उपरिव्यय : खरीदी गई वस्तु को बिक्री करने तक उस पर होने वाला व्यय [ Expenses ], जैसे बिक्री केन्द्र तक लाने में किया जाने वाला व्यय और इसके रख-रखाव पर किए गए व्यय आदि.

4. लागत मूल्य : यह क्रय मूल्य और उपरिव्यय का योग होता है. इस प्रकार लागत मूल्य = क्रय मूल्य + उपरिव्यय

5. लाभ [ Profit ] : यह विक्रय मूल्य - क्रय मूल्य के बराबर होता है. यानी खरीदी गई वस्तु को जिस अधिक मूल्य पर विक्रय किया जाता है, वह अधिक मूल्य लाभ कहलाता है.

6. हानि [ Loss ] : यह क्रय मूल्य - विक्रय मूल्य के बराबर होता है. यदि कोई व्यापारी किसी वस्तु को क्रय मूल्य ( Purchasing Price ) से कम मूल्य पर बेच देता है, तो उसे हानि ( Loss ) होती है.

Note : Profit and Loss की गणना सदैव क्रय मूल्य पर की जाती है. The gain or loss is always reckoned on the cost price.

7. बट्टा [Discount ] हमेशा अंकित मूल्य [ Face Value ] पर गिना जाता है.

8. अंकित मूल्य [ Face Value ] : वस्तु का वह मूल्य जो उस पर अंकित होता है.

आइए अब हमलोग लाभ और हानि ( Profit and Loss )के उपर्युक्त सूत्र ( Formula ) से संबंधित सवाल [ Questions ] देखते हैं. उदाहरण [ Examples ]

1. किसी 1100 रूपये की वस्तु को 120 रूपये में बेचा जाता है. लाभ प्रतिशत की गणना करें. हल : वस्तु का क्रय मूल्य = 100 रूपया तथा विक्रय मूल्य = 120 रूपया इसलिए लाभ = 120-100 = 20 रूपया इसलिए लाभ% = लाभ*100/क्रय मूल्य = 20*100 /100 = 20% [उत्तर]

2. किसी 500 रूपये की वस्तु को 400 रूपये में बेच दी जाती है. हानि प्रतिशत की गणना करें. हल : दिया है, वस्तु का क्रय मूल्य = 500 रूपया तथा विक्रय मूल्य = 400 रूपया इसलिए हानि = 500-400 = 100 रूपया अतः हानि% = 100*100/500 = 20% हानि [उत्तर]

3. 500 रूपये की वस्तु को कितने में बेचा जाए ताकि 20 प्रतिशत लाभ हो ? हल : वि. मू. = [(100+20)/100] * 500 = 600 रूपया [उत्तर]

4. 960 रूपया में किसी वस्तु को बेचेजाने पर 20% का लाभ होता है. वस्तु का क्रय मूल्य क्या है ? हल : क्रय मूल्य = ( 100 ) / (100+20 )] * 960 = 800 रूपया [उत्तर]

5. कोई वस्तु 300 रूपये में बेचे जाने पर 25 % की हानि होती है. वस्तु का क्रय मूल्य क्या है ? हल : क्रय मूल्य = [100 / 100-250] * 300 = [100/75]* 300 = 400 रूपया [उत्तर] **लाभ और हानि के सवाल से संबंधित महत्वपूर्ण ट्रिक [ Profit and Loss Math Trick ]
